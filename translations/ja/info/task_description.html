Stephenのスピーチモジュールは壊れています。 このモジュールは彼の数字の発音を担当しています。 彼はある図に数を全て入力するためにクリックしなければいけません。なので大きな数字のときには長い時間がかかります。 彼のために新しいスピーチモジュールを書くことで、適切にしゃべり数字の処理速度を増加させるようにこのロボットを助けてください。 文字列の全てのワードは正確に一つの空白文字で区切られていなければいけません。 空白に注意してください -- 一つでなく二つの空白を置いたかどうかを確認するのは難しいです。

入力: 数、整数

出力: 数を表現する文字列、文字列

例: 

checkio(4)=='four'
checkio(143)=='one hundred forty three'
checkio(12)=='twelve'
checkio(101)=='one hundred one'
checkio(212)=='two hundred twelve'
checkio(40)=='forty'

どのように使われるか: このコンセプトは音声合成ソフトまたは自動レポートシステムに役に立ちます。 語句に数値を割り当ててこれらの値に基づく反応を取り出すシステムを持つことでchatbotを書くときにも役に立ちます。

事前条件: 0 < number < 1000
