Módulo de fala de Estevão está quebrado. Este módulo é responsável pela pronúncia dos números. Ele precisa clicar para inserir todos os dígitos numéricos em um número, por isso quando há números muito grandes, ele pode levar muito tempo. Ajude o robô a falar corretamente e aumentar sua velocidade de processamento de voz para números escrevendo um novo módulo de voz para ele. As regras para elaboração dos números em Inglês para este problema diferentes: Os algarismos de um a dezenove têm nomes específicos. De vinte e um a noventa e nove, as dezenas e as unidades são unidas por um espaço. Para todos os números de três dígitos, escreve-se o número de centenas em seguida as dezenas e as unidades. Todas as palavras na string devem ser separadas por exatamente um caractere de espaço. Tenha cuidado com os espaços -- é difícil ver se você colocou dois espaços, em vez de um.

Entrada: Um numero inteiro (int).

Saída: A representação do número escrito por extenso na forma de string.

Exemplo: 

checkio(4)=='four'
checkio(143)=='one hundred forty three'
checkio(12)=='twelve'
checkio(101)=='one hundred one'
checkio(212)=='two hundred twelve'
checkio(40)=='forty'

Como é utilizado: Este conceito pode ser útil para o software de síntese de voz ou sistemas de relatórios automáticos. Este sistema também pode ser utilizado quando se cria um modulo de conversação para um computador ou robô capaz de obter respostas com base nesses valores.

Condições prévias: 0 < number < 1000
