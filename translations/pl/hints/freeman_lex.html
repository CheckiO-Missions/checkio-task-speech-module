Nie mam pojęcia, jak zacząć rozwiązywać tę misję.

rozwiązanie, które Panu proponuję jest następujące.

  • Po pierwsze, proszę wprowadzić takie zmiany w początkowych listach, aby każde słowo odpowiadało swojemu indeksowi. Dzięki temu dalsze wyrażenia będą bardziej czytelne.

  • Po drugie, aby rozbić początkową liczbę na listę / krotkę liczb całkowitych, uzupełnionych zerami do długości trzech (maksymalna długość liczby w tej misji).

  • Po trzecie, utworzenie nowej listy ze słowami z początkowych list zamiast odpowiednich cyfr z "złamanej" liczby (biorąc pod uwagę konkretny przypadek z drugą dziesiątką - 10-19).

  • I ostatnie, proszę połączyć tę listę słów (bez pustych elementów) w ciąg znaków z białymi znakami i zwrócić go.

Ok, od czego powinienem zacząć?

Dodajmy do początkowych list puste elementy na początku, aby słowa odpowiadały ich indeksom. Proszę również usunąć HUNDRED - w rozwiązaniu napisze Pan słowo "hundred". Z list tworzę również krotki, ale nie ma to znaczenia - można je zachować jako listy. 

FIRST_TEN = "", "one", "two", "three", "four", "five", "six",\
"seven", "eight", "nine"
SECOND_TEN = "ten", "eleven", "twelve", "thirteen", "fourteen",\
"fifteen", "sixteen", "seventeen", "eighteen", "nineteen"
OTHER_TENS = "", "", "twenty", "thirty", "forty", "fifty", "sixty",\
"seventy", "eighty", "ninety"

Symbol \ jest używany do kontynuowania wyrażenia w następnej linii, jeśli to konieczne.

Proszę mi wyjaśnić, jak przekonwertować liczbę na krotkę liczb całkowitych.

Powinien Pan zacząć od konwersji liczby na ciąg znaków za pomocą wbudowanej funkcji str(). 

str(num)

Teraz proszę uzupełnić ciąg zerami do długości trzech. Dzięki temu dalsze wyrażenia będą prostsze i bardziej czytelne. Proszę to zrobić za pomocą metody string str.zfill(int). 

str(num).zfill(3)

Proszę podzielić ciąg znaków i przekonwertować każdy znak na liczbę całkowitą za pomocą wbudowanej funkcji map(). Zwraca ona obiekt map. 

map(int, str(num).zfill(3))

Będzie to przydatne później, jeśli rozpakujemy ten obiekt do krotki h, t, o - odpowiednio setki, dziesiątki, jedynki. Ponieważ długość ciągu po uzupełnieniu zerami zawsze będzie wynosić 3, zawsze będą Państwo mieli setki i dziesiątki, nawet jeśli ich wartości pozostaną zerami. 

h, t, o = map(int, str(num).zfill(3))

To było szczegółowe! Teraz proszę mi pomóc zbudować nową listę ze słowami zamiast cyfr.

Proszę zadeklarować nową zmienną res jako krotkę słów dla cyfry "setek". Wystarczy użyć zmiennej h jako indeksu elementu krotki FIRST_TEN, ponieważ liczenie setek jest takie samo jak liczenie jedynek - jeden (sto), dwa (sto) itd. Jeśli h równa się 0 pusty ciąg zostanie pobrany z krotki. 

res = FIRST_TEN[h], ...

Teraz należy dodać słowo "hundred", które jest potrzebne tylko wtedy, gdy h (liczba setek) nie jest równa 0. Proszę użyć funkcji bool(), aby wykonać sprawdzenie. 

res = FIRST_TEN[h], "hundred"*bool(h)

Jeśli h jest równe 0, ten element krotki będzie pustym ciągiem.

Jak uzupełnić krotkę dziesiątkami i jedynkami?

Zasadniczo robimy to samo, co w poprzednim kroku z setkami. Jeśli wartość dziesiątki t jest równa 1, oznacza to, że mają Państwo do czynienia z drugą dziesiątką - dziesięć, jedenaście, dwanaście itd. W takim przypadku należy użyć wartości zmiennej ones o jako indeksu dla elementu w krotce SECOND_TEN. 

res += (SECOND_TEN[o],) if t == 1 else ...

Proszę pamiętać, że res jest krotką, więc należy dodać dziesiąte słowo jako element krotki. Dlatego powyższe wyrażenia mogą wyglądać dziwnie.

W innym przypadku, gdy t nie jest równe 1, wystarczy dodać krotkę z dwoma odpowiednimi słowami dla indeksów t i o z krotek OTHER_TENS i FIRST_TEN. 

res += (SECOND_TEN[o],) if t == 1 else (OTHER_TENS[t], FIRST_TEN[o])

Krotka jest zbudowana. Ale jak przekonwertować ją na właściwą linię słów?

Proszę usunąć wszystkie puste ciągi (bez setek, dziesiątek, jedynek) z krotki za pomocą wbudowanej funkcji filter() i funkcji bool jako reguły filtrowania. 

filter(bool, res)

Proszę użyć wyniku filtrowania jako argumentu dla metody str.join(iterable). Proszę połączyć słowa za pomocą białych znaków. 

" ".join(filter(bool, res))

Jak więc powinno wyglądać moje ostateczne rozwiązanie?

Wystarczy dodać końcowe wyrażenia z poprzednich wskazówek, zawinąć kod w wyrażenie funkcji i zwrócić połączony ciąg znaków.

Oto całe rozwiązanie:

FIRST_TEN = "", "one", "two", "three", "four", "five", "six",\
"seven", "eight", "nine"
SECOND_TEN = "ten", "eleven", "twelve", "thirteen", "fourteen",\
"fifteen", "sixteen", "seventeen", "eighteen", "nineteen"
OTHER_TENS = "", "", "twenty", "thirty", "forty", "fifty", "sixty",\
"seventy", "eighty", "ninety"

def checkio(num: int) -> str:
    
    h, t, o = map(int, str(num).zfill(3))
    
    res = FIRST_TEN[h], "hundred"*bool(h)
    res += (SECOND_TEN[o],) if t == 1 else (OTHER_TENS[t], FIRST_TEN[o])
    
    return " ".join(filter(bool, res))

W tym rozwiązaniu nie ma konwersji na ciąg znaków. Oddzielne cyfry są otrzymywane jako iloraz lub przypomnienie z dzielenia. 

FIRST_TEN = ["one", "two", "three", "four", "five", "six", "seven",
"eight", "nine"]
SECOND_TEN = ["ten", "eleven", "twelve", "thirteen", "fourteen", "fifteen",
             "sixteen", "seventeen", "eighteen", "nineteen"]
OTHER_TENS = ["twenty", "thirty", "forty", "fifty", "sixty", "seventy",
             "eighty", "ninety"]
HUNDRED = "hundred"

def checkio(number: int) -> str:

   n = number // 100
   t = [FIRST_TEN[n-1], HUNDRED] if n> 0 else []

   n = (number // 10) % 10
   t += [OTHER_TENS[n-2]] if n> 1 else []

   n = number % (10 if n> 1 else 20)
   t += [(FIRST_TEN+SECOND_TEN)[n-1]] if n> 0 else []

   return ' '.join(t)

W tym rozwiązaniu zastosowano rekurencję i słowa dla pierwszych dwóch dziesięciu połączono w jeden słownik. 

def checkio(number: int) -> str:

    l=["","one","two","three","four","five","six","seven",\
    "eight","nine","ten","eleven","twelve","thirteen",\
    "fourteen","fifteen","sixteen","seventeen","eighteen",\
    "nineteen","twenty"]
    d=dict(enumerate(l))
    d.update({30:"thirty",40:"forty",50:"fifty",60:"sixty",\
    70:"seventy",80:"eighty",90:"ninety"})
    h=number//100
    if h:
        return (d[h]+ " hundred " + checkio(number%100)).strip()
    if number in d:
        return d[number]
    return d[number//10*10]+" "+d[number%10]
